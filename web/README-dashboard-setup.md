# PropertyOps Dashboard Starter Files

1. At the repository root, create the Vite application:
   npm create vite@latest web -- --template react-ts
2. Enter the folder and install dependencies:
   cd web
   npm install
3. Replace the generated `web/src` folder with the `src` folder from this package.
4. Copy `.env` into the `web` folder.
5. Start the dashboard:
   npm run dev

The dashboard requires API CORS to allow `http://localhost:5173`.
